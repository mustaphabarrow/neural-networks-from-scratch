# Neural Networks Notebooks

This folder contains step-by-step Jupyter notebooks documenting my journey building neural networks from scratch.

## Notebooks

| Notebook | Description |
|----------|-------------|
| `01_forward_pass.ipynb` | Single neuron and layer forward pass |
| `02_layers_and_numpy.ipynb` | Multiple layers with NumPy |
| `03_activation.ipynb` | ReLU and Softmax activation functions |
| `04_loss.ipynb` | Categorical cross-entropy loss |
| `05_training.ipynb` | Basic training loop |
| `06_backpropagation.ipynb` | Backpropagation implementation |
| `full_code.ipynb` | Complete neural network implementation |

## How to Use

1. Install dependencies: `pip install numpy matplotlib nnfs`
2. Open notebooks in order
3. Run cells to see outputs
4. Experiment with parameters

## Key Concepts

- Forward propagation
- Activation functions (ReLU, Softmax)
- Loss calculation
- Backpropagation
- Gradient descent
- Multiple optimizers (SGD, Adam, RMSprop)
